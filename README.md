# Coding Test Study (Python)

파이썬 기준 코딩테스트 학습 레포지토리입니다.  
목표는 자주 나오는 알고리즘/자료구조 유형을 개념 + 템플릿 + 문제 패턴 중심으로 정리하는 것입니다.

---

## 공부 주제

1. Python 공통 기초
2. Hash
3. String
4. Priority Queue
5. Binary Search
6. BFS / DFS
7. Union-Find
8. Shortest Path
9. Tree / LCA
10. Square Root Decomposition / Bucket
11. Time Based API 구현
12. 구현 타임어택

---

## 폴더 설명

- `concepts/`  
  주제별 개념 정리 문서

- `templates/`  
  바로 복붙 가능한 Python 템플릿 코드

- `patterns/`  
  문제를 보고 어떤 유형인지 분류하는 기준 정리

- `practice/solved/`  
  푼 문제 기록

- `practice/mistakes/`  
  실수 / 오답 / 재복습 포인트 정리

- `snippets/`  
  파이썬 문법, 자주 쓰는 코드 조각 정리

## 문서 공통 목차

각 주제 문서는 아래 구조를 따릅니다.

1. 왜 필요한가
2. 언제 떠올려야 하는가
3. 핵심 아이디어
4. 시간복잡도
5. Python에서 쓰는 도구
6. 기본 템플릿
7. 자주 하는 실수
8. 대표 문제 유형
9. 확장 개념
10. 한 줄 요약

## 실전 체크 포인트

문제를 보면 먼저 아래를 확인한다.

- 자료구조로 푸는 문제인가?
- 탐색 문제인가?
- 정답을 이분탐색해야 하는가?
- 문자열/파싱 구현인가?
- 그래프 최단거리인가?
- 시간 흐름에 따라 상태를 관리하는가?
